## 🔧 使用方法

### 1. 通过命令接口

```javascript
const cmdInterface = require('./cat-command-interface');

// 处理用户命令
const result = cmdInterface.handleCommand(userId, '健康检查');
console.log(result.content);
```

### 2. 直接使用系统模块

```javascript
const healthWellness = require('./cat-health-wellness');

// 健康检查
const report = healthWellness.healthCheckup(catId);

// 应用互动
healthWellness.applyInteraction(catId, 'play');
```

### 3. 定时任务

```javascript
const cmdInterface = require('./cat-command-interface');

// 每小时执行
const reports = cmdInterface.hourlyHealthCheck();
for (const report of reports) {
  if (report.warnings.length > 0) {
    // 发送警告
  }
}
```
