# 🐱 猫咪养成系统 - 快速安装指南

## 3 步开始养猫！

### 步骤 1: 复制技能文件

```bash
# 技能已经在以下位置：
~/.openclaw/workspace/skills/cat-pet

# 如果是从别人那里获取的，复制到：
cp -r /path/to/cat-pet ~/.openclaw/workspace/skills/
```

### 步骤 2: 重启 OpenClaw

```bash
openclaw gateway restart
```

### 步骤 3: 创建你的猫咪

```
创建猫咪
```
