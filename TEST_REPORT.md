## 🔧 已知问题

### 问题 1: require 缓存

**现象**: 部分测试失败，提示 `getFallbackData is not a function`

**原因**: Node.js require 缓存导致方法未更新

**影响**: 不影响实际功能，仅测试脚本问题

**解决**: 
```javascript
// 清除 require 缓存
delete require.cache[require.resolve('./src/data/error-handler')];
```
